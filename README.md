## API endpoints:

#### GET  
`/api/staff{id}` returns `{name, department}`  
`/api/dailyform{day}` returns `{id, name, room, time_in, time_out, tag, tag_ret, staff.name, staff.department}`  

#### POST

`/api/dailyform/add_entry{entry}` creates new entry on dailyform
